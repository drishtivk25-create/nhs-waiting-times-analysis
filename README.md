# nhs-waiting-times-analysis

## Project Overview
An end-to-end data analysis of NHS England Referral to Treatment (RTT) waiting times across all NHS trusts in England. This project explores where patients are waiting longest, which specialties have the worst backlogs, and how regional performance compares against the NHS 18-week target.

Built to demonstrate a complete analyst workflow: data extraction → cleaning → analysis → SQL exploration → interactive Tableau dashboard.

## Tools & Technologies
- **Python** (pandas, matplotlib, seaborn) — data cleaning & analysis
- **SQL** (SQLite) — data querying & exploration
- **Tableau** — interactive dashboard
- **Data Source** — NHS England open data (RTT Waiting Times, publicly available)

## Key Questions Answered
1. Which NHS trusts have the longest average waiting times?
2. Which medical specialties have the worst backlogs?
3. How do waiting times compare across NHS regions?
4. What percentage of patients are waiting over 18 weeks (the NHS target)?
